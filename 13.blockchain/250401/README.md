# 이번 시간에는 뭐할까?

## 수업 핵심 주제

1. **블록체인은 비효율적이다**
2. **그래서 블록체인은 아름답다**

“네??” 싶죠?

오늘은 **블록체인을 소개하는 시간**이기도 하지만,  
무엇보다도 **여러분에게 ‘납득’을 시키는 시간**이기도 합니다.

## 오늘의 목표

블록체인을 배우며, 우리가 반드시 고민할 질문들:

- 우리는 **왜 블록체인을 공부해야 할까?**
- 우리는 **어떤 시각으로 블록체인을 바라봐야 할까?**
- 그래서 우리는 **무엇을 직접 구현하고 경험하게 될까?**

## 수업 방향

> “우리는 단순히 기술을 배우는 것이 아닙니다.”

- 기술의 **철학적 배경**을 이해하고,
- **구조와 원리**를 설계하고,
- **직접 체험**하며 납득하는 것까지.
