# 백엔드 – 6월 9일 킥오프 전 준비 플랜

## 전제 조건

- 실질적인 복습 가능 날짜는 **6월 2일, 4일, 5일 단 3일**
- 목표는 **Node.js 기반 구조화 백엔드 설계 감각 체득**
- 단순 문법 연습이 아님
  → 팀 협업에서 **백엔드 역할을 맡을 수 있을 정도의 구조 이해**가 핵심

## 전체 목표

> **Counter 기능을 구현하며, 설계 중심 백엔드 감각을 익히는 것**

- **HTTP 요청 처리 흐름**
- **비동기 로직과 에러 처리 구조**
- **계층 분리 (Controller / Service / Model / Middleware)**
- **의존성 주입 사고방식(DI 감각)**
